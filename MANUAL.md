# Vitalink Wiki - 用户手册

## 概览
Vitalink 是由Rutsunine主创的个人OC知识库。它支持富文本编辑、实时协作（开发中）和版本控制功能。

## 用户角色
- **访客 (Guest)**: 可以浏览公开文章。
- **普通用户 (User)**: 可以浏览文章，需申请编辑权限。
- **编辑 (Editor)**: 可以创建和修改文章。
- **管理员 (Admin)**: 拥有系统完整权限。

## 快速上手

### 1. 注册与登录
- 点击顶部导航栏的 **LOGIN (登录)**。
- 如果没有账号，点击 **Register (注册)**。
- 填写您的姓名、邮箱和密码完成注册。

### 2. 浏览文章
- 点击导航栏的 **WIKI** 或在主页进行搜索。
- 文章页面采用无干扰的沉浸式阅读设计。

### 3. 创建与编辑内容
- 访问想要创建的 URL（例如 `/wiki/new-topic`）或在主页点击 **开始写作 (Start Writing)**。
- 点击右上角的 **编辑文章 (EDIT ARTICLE)** 按钮。
- 使用编辑器撰写内容，支持 Markdown 和富文本格式。
- 点击 **保存更改 (SAVE CHANGES)** 发布。系统会自动创建一个新版本。

### 4. 版本历史
- 在文章页面点击 **History (历史记录)**（如可用）查看过往版本。
- 您可以随时回退到旧版本。

## 编辑贴士
- **格式化**: 支持标准 Markdown 语法快速排版。
- **多媒体**: 支持嵌入图片和视频（功能完善中）。
- **协作**: 支持多人同时编辑（功能开发中）。

## 常见问题
- **登录失败**: 请确认邮箱和密码输入正确。
- **保存错误**: 请检查网络连接是否正常。
